# 📖 153 말씀암송 (Psalms 121)

시편 121편을 쉽고 재미있게 암송할 수 있도록 돕는 웹 애플리케이션입니다.

## ✨ 주요 기능

1.  **다국어 지원**:
    *   🇰🇷 한글 (개역개정)
    *   🇺🇸 영어 (NIV)
    *   🇰🇷🇺🇸 한글 + 영어 대조 모드
2.  **단계별 암송**:
    *   전체 읽기 모드
    *   절별 선택 암송 (1절 ~ 8절)
    *   빈칸 채우기 (난이도 조절: 20%, 50%, 80%)
3.  **다양한 학습 방식**:
    *   ⌨️ 직접 타이핑하여 채우기
    *   👆 클릭하여 정답 확인하기
4.  **반응형 디자인**: PC와 모바일 모두에서 최적화된 화면을 제공합니다.

## 🛠 사용 기술

*   HTML5
*   CSS3 (Flexbox, Animations)
*   Vanilla JavaScript

*   Google Fonts (Jua)
